About Reese’s TV

Reese’s TV is a lightweight, static streaming site hosted on GitHub Pages. Videos are embedded from Google Drive.

How it works

  1. Drop your video metadata into videos.json.
  2. Each entry uses the Google Drive file ID for embedding.
  3. GitHub Pages hosts this site—no backend required.

Made to match the playful Reese’s branding 💙
